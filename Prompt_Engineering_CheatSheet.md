# 🧠 Prompt Engineering Cheat Sheet

## 🛠️ Estructura Básica
```
[Contexto o Rol] + [Instrucción clara] + [Formato deseado]
```
Ejemplo:
> *Eres un chef profesional. Sugiere una receta saludable con aguacate, en formato de lista paso a paso.*

---

## 🎭 Roles y Contexto
- "Actúa como un experto en ciberseguridad..."
- "Eres un ingeniero de software con 10 años de experiencia..."
- "Asume el rol de un historiador especializado en la antigua Grecia..."

---

## 🔍 Tipos de Prompts

### 1. **Instrucciones Claras**
> Resume este artículo en 3 puntos clave.

### 2. **Ejemplos para guiar**
> Convierte este texto a un tweet. Ejemplo: “Los datos muestran que…”

### 3. **Formato específico**
> Devuélveme la respuesta en formato JSON.

### 4. **Iterativo**
> Dame 3 ideas. Luego refina la 2da.

### 5. **Comparativo**
> Compara MySQL y PostgreSQL en velocidad, funciones y escalabilidad.

---

## 📚 Estructuras útiles

| Acción     | Ejemplo                                                        |
|------------|----------------------------------------------------------------|
| Explica    | Explica cómo funciona una red neuronal a un principiante.     |
| Resume     | Resume el texto en 100 palabras.                              |
| Lista      | Haz una lista de herramientas para DevOps.                    |
| Traduce    | Traduce al inglés: "Hola, ¿cómo estás?"                       |
| Reescribe  | Reescribe el texto como si fuera una novela de misterio.      |
| Genera     | Genera un título llamativo para este post de blog.            |

---

## ✨ Tips para Prompts Efectivos
- ✅ Sé específico y claro.
- ✅ Da ejemplos si el formato importa.
- ✅ Define el público objetivo.
- ✅ Usa iteraciones para refinar.
- ❌ Evita vaguedades como “hazlo mejor”.
